### 技术架构
- 前端：Next.js 15 App Router + React 19 + Tailwind CSS v3 + shadcn/ui
- Markdown：react-markdown + remark-gfm + rehype-highlight + 自定义 Obsidian 语法处理器
- 数据：gray-matter 解析 frontmatter，JSON 文件存储评论和浏览数据
- 动画：framer-motion 滚动动画
- 图标：lucide-react

### 目录结构
- `content/posts/` — Obsidian 兼容的 Markdown 博客文章
- `src/app/` — Next.js App Router 页面和 API 路由
- `src/components/` — UI 组件、布局组件、博客组件、主页组件
- `src/lib/` — 工具函数、文章处理、Obsidian 兼容、数据存储
- `data/` — 评论和浏览数据（运行时自动创建）

### 运行方式
- 开发：`npx next dev`
- 构建：`npx next build`
- 生产：`npx next start`
- 文章添加：在 `content/posts/` 目录创建 .md 文件即可
