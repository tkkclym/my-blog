# 2026-07-06 工作日志

## 个人技术博客项目搭建

### 完成内容
- 搭建了基于 Next.js 15 + React 19 + TypeScript + Tailwind CSS + shadcn/ui 的现代个人博客系统
- 绿色科技主题，深色模式默认，glassmorphism 玻璃拟态设计
- 完整的 Obsidian Markdown 兼容层（双链、嵌入、Callout、高亮等语法）
- 博客系统：文章列表（搜索/分类/标签筛选）、文章详情、Markdown 渲染
- 评论功能（API + JSON 本地存储）、浏览次数统计
- 个人主页：Hero 动画、技能展示、时间线履历、最新文章
- 关于页面：个人简介、技术栈、联系方式
- 3 篇示例 Obsidian 格式博客文章
- 生产构建成功，所有页面和 API 验证通过

### 技术架构
- 前端：Next.js 15 App Router + React 19 + Tailwind CSS v3 + shadcn/ui
- Markdown：react-markdown + remark-gfm + rehype-highlight + 自定义 Obsidian 语法处理器
- 数据：gray-matter 解析 frontmatter，JSON 文件存储评论和浏览数据
- 动画：framer-motion 滚动动画
- 图标：lucide-react

### 目录结构
- `content/posts/` — Obsidian 兼容的 Markdown 博客文章
- `src/app/` — Next.js App Router 页面和 API 路由
- `src/components/` — UI 组件、布局组件、博客组件、主页组件
- `src/lib/` — 工具函数、文章处理、Obsidian 兼容、数据存储
- `data/` — 评论和浏览数据（运行时自动创建）

### 运行方式
- 开发：`npx next dev`
- 构建：`npx next build`
- 生产：`npx next start`
- 文章添加：在 `content/posts/` 目录创建 .md 文件即可
